Intelligent Geographic Information Retrival Using Ontology

The java programs helps to find the shortest time path between two locations andplots it on a google map.
A RDF data base has been used to store the information by creating an ontology table. The .ttl files stores the required geological information.Parliament data base has been used along with SPARQL for information retrival. The programs requiers Jena api, Google maps api and java windows builder (for GUI). Currently the data is stored for a part of NIT, Nagpur college.
